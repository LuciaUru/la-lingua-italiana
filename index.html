Regole di pronuncia

C + e/i = "ch" · ciao → "chao" · cinema → "chinema"
C + a/o/u = "k" · casa → "kasa" · come → "kome"
G + e/i = "y" suave · gelato → "yelato" · giro → "yiro"
Gli = "ll" · figlio → "fillo" · foglia → "folla"
Gn = "ñ" · gnocchi → "ñoki" · bagno → "baño"
Z = "ts" · pizza → "pittsa" · zero → "tsero"
H = siempre muda · hanno → "anno" · ho → "o"
R = vibrante (como el español rioplatense) · Roma · amore
