Los fundamentos del italiano para principiantes

I. Los artículos determinativos

En italiano, el artículo concuerda en género y número con el sustantivo.
GéneroSingularPlural
Masculinoil / lo / l'i / gli
Femeninola / l'le
il libro (el libro) · la casa (la casa) · l'uomo (el hombre)

II. El verbo essere — ser/estar

Essere es uno de los verbos más usados. Se usa tanto para "ser" como para "estar".
PronombreForma
io (yo)sono
tu (vos/tú)sei
lui/lei (él/ella)è
noi (nosotros)siamo
voi (ustedes)siete
loro (ellos)sono

III. El verbo avere — tener/haber

Reglas de pronunciación

C + e/i = suena como "ch" · ciao = "chao"
C + a/o/u = suena como "k" · casa = "kasa"
G + e/i = suena como "y" suave · gelato = "yelato"
Gli = suena como "ll" · figlio = "fillo"
Gn = suena como "ñ" · gnocchi = "ñoki"
Z = suena como "ts" · pizza = "pittsa"
